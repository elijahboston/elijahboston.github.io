Projects

Squid Knocker (beta)

A news reader for music. Squid Knocker scans music blogs and lets you listen to the tracks they post. This project is pretty hot off the press and still has a long way to go, but it's also come very far and I'm quite proud of it.

On the backend I'm using Django as a REST service with authorization through Facebook.

This allows the frontend to be a lightweight single-page application that's entirely dynamic and event based, using Backbone.js and modular Javascript (Require.js).

On the server architecture side of things, using nginx and uwsgi resulted in some pretty awesome performance benchmarks (~9mil hits/day).

Interstate Batteries

As the primary frontend/backend developer I was responsible for adding interaction to templates using Backbone.js. I worked on a backend component that combined Interstate Batteries' store location finder and the customer's phone GPS to display the stores closest to the customer.

Cheaper Than Dirt

Listunes

2010-2013

My first attempt at building a web app based around data scraping and using 3rd party API's. There were 3 main components: a modified version of Feedjack which retrieved new blog posts and flags them for parsing, a script that parsed each post for media content (MP3's and SoundCloud), and a frontend that displayed the latest media that had been found.
